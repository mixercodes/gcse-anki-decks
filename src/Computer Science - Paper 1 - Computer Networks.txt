Basic	What is the difference between a LAN and a WAN?	<b>LAN (Local Area Network):</b> connects devices within a small geographical area (one building or site). Owned and managed by the organisation. Uses Ethernet cables or Wi-Fi. Fast, cheap to maintain.<br><b>WAN (Wide Area Network):</b> connects LANs across a large geographical area (countries, globally). Uses leased telephone lines, fibre optic, or satellite links. The Internet is the largest WAN — not owned by any single organisation.	Computer Science::Paper 1::Computer Networks
Basic	Compare client-server and peer-to-peer (P2P) networks.	<b>Client-server:</b> a central server provides resources/services; clients request them. Centralised security, easier backups, scalable. Better for businesses and schools.<br><b>Peer-to-peer (P2P):</b> all devices (peers) are equal and can share resources directly. No central server. Cheaper to set up, harder to secure and manage. Better for small home networks and file-sharing (e.g. BitTorrent).	Computer Science::Paper 1::Computer Networks
Cloze	Network topologies: {{c1::star}} — all devices connect to a central switch; fast, easy fault isolation, but expensive (more cable) and switch failure brings down the whole network. {{c2::Mesh}} — devices connect to multiple others; highly resilient but very expensive and complex to cable.		Computer Science::Paper 1::Computer Networks
Basic	What is the function of each: NIC, switch, router, WAP?	<b>NIC (Network Interface Card):</b> hardware that enables a device to connect to a network (wired or wireless).<br><b>Switch:</b> connects devices on a LAN; directs data only to the intended recipient using MAC addresses (more efficient than a hub).<br><b>Router:</b> connects different networks (e.g. home LAN to the Internet); routes data packets between networks using IP addresses.<br><b>WAP (Wireless Access Point):</b> connects wireless devices to a wired network using Wi-Fi (radio waves).	Computer Science::Paper 1::Computer Networks
Cloze	An {{c1::IP address}} is a logical numerical label assigned to a device on a network — it can change. A {{c2::MAC address}} is a unique 48-bit physical address permanently assigned to a NIC by the manufacturer (shown as 6 hex pairs, e.g. A1:B2:C3:D4:E5:F6) — it cannot normally change. {{c3::DNS}} (Domain Name System) translates {{c4::domain names}} (e.g. google.com) into {{c5::IP addresses}}.		Computer Science::Paper 1::Computer Networks
Cloze	Key network protocols: {{c1::HTTP}} — transfers web pages (unencrypted); {{c2::HTTPS}} — HTTP encrypted with {{c3::TLS}}; {{c4::FTP}} — transfers files between systems; {{c5::SMTP}} — sends email; {{c6::IMAP/POP3}} — retrieves email; {{c7::TCP/IP}} — reliable data delivery over the Internet (TCP reassembles packets; IP handles routing).		Computer Science::Paper 1::Computer Networks
Basic	What is the difference between the Internet and the World Wide Web?	<b>The Internet</b> is the global physical infrastructure — a network of interconnected networks (cables, routers, servers). It hosts many services.<br><b>The World Wide Web (WWW)</b> is one service that runs <i>over</i> the Internet — a collection of web pages stored on web servers and accessed via browsers using HTTP/HTTPS.<br>The Internet also hosts: email (SMTP/IMAP), FTP, VoIP, streaming, online gaming.	Computer Science::Paper 1::Computer Networks
Basic	What is a network protocol and why is it needed?	A protocol is an agreed set of rules governing how data is transmitted between devices on a network. Protocols ensure that devices from different manufacturers running different software can communicate reliably.<br><b>Without protocols:</b> incompatible systems could not understand each other's data (like people speaking different languages with no translation rules).	Computer Science::Paper 1::Computer Networks
Basic	What is the difference between HTTP and HTTPS?	<b>HTTP:</b> HyperText Transfer Protocol — sends web data as <b>plain text</b>. Can be intercepted and read by attackers (man-in-the-middle attack).<br><b>HTTPS:</b> HTTP Secure — encrypts all data using <b>TLS (Transport Layer Security)</b>. A padlock appears in the browser. Essential for login pages, banking, any site handling sensitive data.<br>All modern websites should use HTTPS; HTTP is considered insecure.	Computer Science::Paper 1::Computer Networks
Cloze	Wi-Fi advantages over Ethernet: {{c1::mobility}} (devices can move around), {{c2::easier}} to add new devices, no {{c3::physical cables}} needed. Disadvantages: {{c4::slower}} speeds, less {{c5::secure}} (signals broadcast and can be intercepted), more {{c6::interference}}, higher {{c7::latency}}.		Computer Science::Paper 1::Computer Networks
Basic	Compare star and mesh network topologies in detail.	<b>Star topology:</b> each device has its own dedicated cable to a central switch. Fast data transfer, easy to diagnose faults (one cable failure only affects one device), easy to add new devices. Disadvantage: switch failure = whole network down; more cable needed than mesh for small networks.<br><b>Mesh topology:</b> devices connect directly to multiple other devices. <b>Full mesh</b> — every device connected to every other; maximum resilience but extremely expensive and complex to cable. <b>Partial mesh</b> — only some devices have multiple connections; cheaper compromise. No single point of failure (unlike star). Used where reliability is critical (e.g. the Internet backbone).	Computer Science::Paper 1::Computer Networks
Basic	What is the role of DNS (Domain Name System)?	DNS acts like a phone book for the Internet. When a user types a domain name (e.g. www.bbc.co.uk), the DNS server translates it into the corresponding <b>IP address</b> (e.g. 212.58.244.22) so the router can direct the request to the correct server. Without DNS, users would have to memorise IP addresses for every website.	Computer Science::Paper 1::Computer Networks
Cloze	IPv4 addresses are {{c1::32}}-bit numbers written as four {{c2::denary}} numbers separated by dots (e.g. 192.168.1.1). IPv6 uses {{c3::128}}-bit addresses written in hexadecimal (e.g. 2001:0db8::1). IPv6 was introduced because IPv4 only supports about {{c4::4.3 billion}} unique addresses — not enough for all modern devices.		Computer Science::Paper 1::Computer Networks
Basic	What factors affect network performance?	<b>Bandwidth:</b> the maximum amount of data that can be transmitted per second (measured in Mbps/Gbps). Higher bandwidth = faster data transfer.<br><b>Latency:</b> the delay between data being sent and received (measured in milliseconds). High latency causes lag — especially noticeable in video calls and gaming. Caused by distance, congestion, and processing at routers.<br><b>Number of connected devices:</b> more devices sharing the same network reduces the bandwidth available to each — performance degrades under heavy use.<br><b>Transmission medium:</b> wired connections (fibre optic &gt; copper Ethernet) are faster and more reliable than wireless (Wi-Fi). Fibre optic uses light pulses; copper uses electrical signals; wireless uses radio waves.<br><b>Interference and signal loss:</b> wireless signals degrade with distance and physical obstacles (walls). Copper cables are susceptible to electromagnetic interference; fibre optic is immune.	Computer Science::Paper 1::Computer Networks
Cloze	Network performance factors: {{c1::bandwidth}} is the maximum data per second (Mbps/Gbps) — higher is faster. {{c2::Latency}} is the delay in milliseconds — caused by distance, congestion, and router processing. More {{c3::connected devices}} share bandwidth, slowing the network. {{c4::Fibre optic}} is faster than {{c5::copper}} Ethernet, which is faster than {{c6::wireless (Wi-Fi)}}. Wireless suffers from {{c7::interference}} and {{c8::signal loss}} with distance; fibre optic is immune to {{c9::electromagnetic interference}}.		Computer Science::Paper 1::Computer Networks
